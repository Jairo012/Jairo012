The objective of this article is to use the ant colony optimization algorithm to address the order picking problem in a warehouse. In addition, the different types of algorithms that have been used over the years to solve this problem are expanded on the traveling agent problem, the ACO, its foundations and its different applications. As a result of this research, it is possible to modify the ACO algorithm for TSP so that it can be used for the order picking problem by entering different orders and obtaining a minimum path that includes the order in which the orders must be picked and the distance involved.
